# Style Library (extended)

A broad, independent database of named looks — separate from the project house DNA in `style-database.md`. Each = a paste-ready **TOKEN BLOCK**. Lock one (or blend two) and reuse byte-identical across a sequence. Blend syntax: `Profile A × Profile B (60/40)`.

> All references describe *techniques and aesthetic traditions* for prompting, not the reproduction of any specific copyrighted work or artist's protected output.

---

## 7. MOOD PRESETS (fast blends)

## Usage
1. Pick/blend a profile. 2. Paste its TOKEN BLOCK as the locked style in the master storyboard. 3. Append character DNA. 4. Reuse unchanged across every panel/shot. For video, add ONE camera move + the audio routing per shot.
